# Alpha Shop - React

(./public/image/Alpha-shop.png)

## 介紹

以 React 進行開發，並以 CSS sass 進行 CSS 樣式撰寫。

完成的元件包含：

- Header
- Main (Register & Cart)
- Footer

### 使用指南

1. 將專案 clone 至本地，至終端機輸入：

   ```
   https://github.com/HelloMapleBoy/ALPHA-Shop.git

   ```

2. 執行專案，輸入：

   ```
   npm start

   npm install node-sass

   ```

3. 瀏覽器 "http://localhost:3000" ，可瀏覽此專案。

4. 停止使用，至終端機：
   ```
   ctrl + c
   ```
